# Computer Science Program

## Purpose <br/>
The problems for this course were to utilize different data structures and the algorithms to traverse them in order to create a system that would quickly and efficiently
comb through hundred or thousands of entries of data and add, delete, or retrieve said data. We utilized structures like a common vector structure, hash map, and binary tree.
In the end for this use the most efficient was the binary search tree, primarily for its ease of searching and ability to cut a tree in half for each level it traverses. 
<br/> <br/>
## Approach <br/>
To approach the problem I took into consideration all the structures we used and weighed the options for which would work properly in this case. I was initially torn between
using a hash set or hash map, and a binary search tree. I knew that a binary search tree on average would perform a search better therefore I went with that for the data structure
to handle our school courses. <br/> <br/>
